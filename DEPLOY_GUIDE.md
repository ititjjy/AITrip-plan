# 行程规划应用 - 阿里云百炼平台部署指南

## 应用信息
- 应用名称：行程规划 Trip Planner
- 技术栈：React + TypeScript + Express + SQLite
- 部署包大小：约 13MB

## 部署步骤

### 1. 登录阿里云百炼平台
访问：https://bailian.console.aliyun.com/
使用阿里云账号登录

### 2. 创建应用
- 点击左侧菜单 "应用管理" → "我的应用"
- 点击 "创建应用"
- 选择部署方式：**静态网站** 或 **Web 应用**
- 应用名称：trip-planner
- 地域：选择离你最近的地域（如 华东2（上海））

### 3. 上传部署包
- 在应用详情页找到 "部署" 选项卡
- 点击 "上传文件"，选择本目录下的 `deploy-package.zip`
- 等待上传完成

### 4. 配置运行环境
#### 环境变量（必需）：
```
NODE_ENV=production
PORT=8080
```

#### 启动命令：
```
npm install --production && npm run start
```

### 5. 配置域名（可选）
- 百炼会自动分配一个测试域名
- 如需自定义域名，可在 "域名管理" 中绑定自己的域名

## 注意事项

1. **数据库**：应用使用 SQLite，数据文件位于 `server/data/` 目录
2. **HTTPS**：百炼平台自动提供 HTTPS 支持
3. **访问地址**：部署成功后会在控制台显示公网访问地址

## 本地开发调试
如需本地调试，运行：
```bash
npm run dev:all
```
